# DailyNews
Assessment for Smart Dubai

Code Setup Instructions after clone / download
----
1. copy the directory path of the file and Run this command in terminal "cd {file_path}"
2. "pod install"
3. open the "DailyNews.xcodeworkspace"

External Libraries Used
----
1. AlamoFire (Handle all the network operations)
2. KingFisher (Handle all the image fetching and caching operations)


Project Structure
----
1. This Project uses simple low level MVC Design patterns with many reusable componenets
2. This Project uses custom cells
